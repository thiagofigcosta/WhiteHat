HOW TO RUN
-------------------------------------------------------
files in /win32
*Windows: Just run "GAME MAKER - whitehat.exe"
*Linux: Run "GAME MAKER - whitehat.exe" using wine

-------------------------------------------------------
HOW TO USE
-------------------------------------------------------
Open the program
Click "Novo Jogo"
Play

-------------------------------------------------------
CONTROLS
-------------------------------------------------------
"A"                     - Change Gun
"S"                     - Shoot
Arrow Keys              - Movement, Jump, Interaction
Double Click Arrow      - Sprint
F1                      - White Screen
ESC                     - Exit, Exit white screen
F12                     - Cheats and NA'VIgation
